Porous Media adalah suatu medium yang terdiri dari bagian padat(skeleton) dan kosong, atau porous media adalah suatu media yang memiliki banyak pori-pori. Pori-pori pada Porous media memungkinkan fluida untuk mengalir melaluinya. Aliran fluida pada Porous media dijelaskan oleh hukum Darcy dan persamaan Kontinuitas. Fluida pada kasus ini dapat berupa cairan maupun gas, bahkan panas juga dapat ditinjau sebagai fluida apabila merambat melalui aliran gas. Untuk menjelaskan bagiamana aliran panas pada Porous media akan dibangun persamaan untuk memodelkannya. Pembuatan model matematika diawali dari ilustrasi berikut.
